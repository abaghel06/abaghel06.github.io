Personal Project

Food Recommendation System

I wanted to understand how vector search actually works, so I built something I'd use: a food recommendation engine where you can ask questions like "what's nutritionally similar to chicken breast?" or "what gives me more iron?" — and get real answers, not keyword matches.

The search layer uses embeddings and ChromaDB. The Q&A layer is a RAG pipeline over a nutritional knowledge base — so answers come with sources, not hallucinations.
